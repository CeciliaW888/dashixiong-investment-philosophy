## 💡 核心概念

### 底层逻辑
支撑一个现象运行的根本原因和机制。大师兄说：「如果没有底层逻辑的，那何止会垮一波？」

### 历史型定价
用历史通胀和货币供应量衡量资产的真实价值，而不是只看名义价格。

### 三分法
大师兄的风险配置方法：将投资分成三部分，应对不同经济场景。

### 长期持有
基于底层逻辑的理解，在相对低点进入，长期持有。不因短期波动而动摇。

## 📖 推荐阅读

**完整书单（按主题分类）：** [推荐书籍 →](recommended-reading.md)

### 前置阅读（开课前）
- 《经济学》— 萨缪尔森（至少前3章）——「历史上最牛的一本经济学著作」
- 《国富论》— 亚当·斯密（了解市场经济基础）
- 《通往奴役之路》— 哈耶克（理解奥地利学派）

### 伴读推荐（学习过程中）
- 《了凡四训》— 袁了凡（「第一章你要读100遍」）
- 《道德经》— 老子（课程中引用最多的经典）
- 《时运变迁》— 保罗·沃尔克 / 行天丰雄（理解货币博弈）
- 《芯片战争》— Chris Miller（「看完这本书你就知道芯片是什么东西」）
- 《石油！石油！》— Leonardo Maugeri（「大家可以买这本书来看一下」）
- 巴菲特致股东的信（「全部都是这一条——消费现金流」）
