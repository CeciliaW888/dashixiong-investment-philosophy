## 💡 核心概念

### 底层逻辑
支撑一个现象运行的根本原因和机制。大师兄说：「如果没有底层逻辑的，那何止会垮一波？」

### 历史型定价
用历史通胀和货币供应量衡量资产的真实价值，而不是只看名义价格。

### 三分法
大师兄的风险配置方法：将投资分成三部分，应对不同经济场景。

### 长期持有
基于底层逻辑的理解，在相对低点进入，长期持有。不因短期波动而动摇。
